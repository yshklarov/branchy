# branchy

A Java implementation of [red-black trees](https://en.wikipedia.org/wiki/Red%E2%80%93black_tree).
Please note: it's still incomplete! The node-removal code does not work yet.

To build:

    $ ant build

To test:

    $ java -jar dist/lib/branchy-<date>.jar
